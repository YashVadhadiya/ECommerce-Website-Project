Creative and Beautiful WordPress Theme with a Large Header

Description

Store is the Last eCommerce WordPress theme you will Ever need. It is loaded with tons of Features including Multiple Layouts, Featured Product Showcase Areas, Fully Configurable Sidebar width, Advanced Menu and Navigation system, Featured Images, Slider with 3D Effects, Google Fonts, High Quality Animation throughout the theme without compromising on the Professionalism. Store is Built using Bootstrap 3 and Font Awesome. And yes, this theme is 100% Responsive and will work on all your devices from Mobiles, Tablets to Desktops. Demo Here: http://demo.inkhive.com/store-pro/
